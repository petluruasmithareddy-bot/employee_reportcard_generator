#  Multiplication Table Generator (Python)

A simple Python program that generates the multiplication table for a given number. This project is beginner-friendly and helps in understanding loops, user input, and basic arithmetic operations.


##  Project Description

This program takes a number as input from the user and prints its multiplication table from 1 to 10.

It is a great starting project for learning:

* Loops (`for`, `while`)
* User input handling
* String formatting (f-strings)
* Basic mathematical operations

##  Features

*  Takes user input
*  Uses loop to generate table
*  Displays formatted multiplication table
*  Easy to understand and beginner-friendly

##  Technologies Used

* Python 3
* No external libraries required

##  How to Use

1. Run the program
2. Enter a number when prompted
3. View the multiplication table from 1 to 10

##  Future Improvements

* Allow custom range (e.g., up to 20 or user-defined)
* Generate multiple tables at once
* Add GUI using Tkinter
