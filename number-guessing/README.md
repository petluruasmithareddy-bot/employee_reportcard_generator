#  Number Guessing Game (Python)

A simple and fun command-line game where the computer randomly selects a number, and the player tries to guess it in the fewest attempts possible.

##  Project Description
This is a beginner-friendly Python project that demonstrates core programming concepts like:

* User input
* Conditional statements
* Loops
* Random number generation

The program generates a random number between 1 and 100, and the user has to guess it. After each guess, the program gives feedback whether the guess is too high or too low.

##  Features

* Random number generation using Python
* Interactive user input
* Hint system (Too High / Too Low)
* Attempt counter
* Simple and clean logic

## Technologies Used

* Python 3
* Built-in `random` module

##  How to Play

1. Run the program
2. Enter your guess when prompted
3. The program will tell you:

   * "Too High" 
   * "Too Low" 
4. Keep guessing until you find the correct number
5. Try to finish in minimum attempts!
