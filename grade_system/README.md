# Student Grade System (Python)

A simple Python program that calculates total marks, average, and assigns a grade based on user input. This project demonstrates the use of functions and basic programming concepts.


##  Project Description

The Student Grade System takes marks of multiple subjects as input from the user, calculates the total and average, and assigns a grade (A/B/C/Fail) based on the average score.

This project is designed to help beginners understand:

* Functions (modular programming)
* Lists
* Loops
* Conditional statements
* User input handling


##  Features

*  Accepts marks for multiple subjects
*  Calculates total marks
*  Computes average
*  Assigns grade based on performance
*  Uses separate functions for clean code


##  Technologies Used

* Python 3
* No external libraries required


##  Grading Criteria

* 90 and above → Grade A
* 75 to 89 → Grade B
* 50 to 74 → Grade C
* Below 50 → Fail

##  Project Structure

* `calculate_total()` → Calculates total marks
* `calculate_average()` → Computes average
* `assign_grade()` → Determines grade
