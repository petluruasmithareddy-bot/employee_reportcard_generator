# Simple Calculator (Python)

A basic command-line calculator built using Python that performs fundamental arithmetic operations like addition, subtraction, multiplication, and division.


##  Project Description

This project is a beginner-friendly implementation of a calculator that allows users to perform mathematical operations through a simple menu-driven interface.

It demonstrates core Python concepts such as:

* Functions
* Conditional statements
* User input handling
* Basic error handling

##  Features

*  Addition
*  Subtraction
*  Multiplication
*  Division
*  Handles division by zero
*  Clean and modular code using functions
  
## Technologies Used

* Python 3
* No external libraries required

##  How to Use

1. Run the program
2. Choose an operation:

   * 1 → Addition
   * 2 → Subtraction
   * 3 → Multiplication
   * 4 → Division
3. Enter two numbers
4. View the result instantly


##  Future Improvements
