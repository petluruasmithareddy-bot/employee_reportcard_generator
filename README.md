# employee_reportcard_generator

## Overview
This project is a simple Python-based **Employee Report Card Generator** that creates and displays structured employee details such as:
- Full name
- Age
- Experience
- Position and salary
- Department details (extracted from employee code)

It also demonstrates how encoded employee data can be processed using string operations.

##  Features
- Generates a complete employee report card
- Combines first and last names into a full name
- Displays employee age and experience
- Formats employee details using f-strings
- Extracts key details from employee code:
  - Department (e.g., DEV)
  - Year (e.g., 2026)
  - Initials (e.g., JD)
##  Concepts Used
- String Concatenation (`+`)
- f-Strings (formatted output)
- Type Casting (`str()`)
- String Slicing (`[start:end]`)
- Variable Manipulation
